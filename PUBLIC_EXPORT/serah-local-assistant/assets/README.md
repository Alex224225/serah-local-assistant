# Assets

This folder is reserved for diagrams, architecture visuals, and screenshots.

## What Would Go Here

- Architecture diagrams (scanner → opportunity engine → Claude audit → human decision)
- Opportunity engine packet flow diagram
- Tier system visual
- Sample dry-run session output (redacted to remove local paths)
- Agent test result screenshots (console output only, no account UI)

## What Will NOT Go Here

- Screenshots of any brokerage interface
- Screenshots showing account numbers, balances, or positions
- Any image containing a local machine path
- Any image that could identify a personal account

## Status

No assets have been added yet. The project is currently documentation-only for
the public export. Diagrams will be added in a future iteration.

If you are reviewing this repository and would like to see specific diagrams or
flow illustrations, please open an issue and describe what would be most useful.
